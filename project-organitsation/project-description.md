#  Project Idea: McLaren Senna Comparison Tool

##  Overview

This project is a JavaScript-based web application that compares the performance of the McLaren Senna with other high-speed sports cars.

The main goal is to create an interactive bar chart that visually shows the differences between the McLaren Senna and another selected car.

---

##  Objective

Users should be able to:

- View the McLaren Senna’s performance statistics by default.
- Select another fast car from a dropdown menu.
- See both cars displayed together in a bar chart.
- Clearly recognize the differences in performance values.

---

##  Compared Statistics

The comparison will include:

- Horsepower (HP)
- Top Speed (km/h)
- 0–100 km/h acceleration (seconds)

---

##  Cars Available for Comparison

Example cars that can be selected:

- Bugatti Chiron  
- Koenigsegg Jesko  
- Ferrari LaFerrari 
- Porsche 918 Spyder
- McLaren P1

---

##  How It Works (Concept)

1. When the website loads, the McLaren Senna’s data is shown in a bar chart.
2. A dropdown menu allows the user to choose another car.
3. After selecting a car:
   - The second car’s statistics are added to the chart.
   - The bar chart updates dynamically.
4. The user can visually compare both vehicles.

---

##  Technologies Used

- HTML for structure  
- CSS for styling  
- JavaScript for logic and interactivity
