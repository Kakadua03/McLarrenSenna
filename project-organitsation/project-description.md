# **Website Project Description**

## **1. Project Title**

* Discover the McLaren Senna

---

## **2. Project Idea**

* This website is about the McLaren Senna, a sports car of the famous brand McLaren.

  Content:
    - Overview
    - Data
    - History
    - Images
    - (Imprint)
  

  This topic is interesting because the McLaren Senna is a unique and advanced car that shows how modern technology and design can achieve high performance.

---

## **3. Target Audience**

* The website is for enthusiasts of motorsport, sports cars or the brand McLaren. It will give them information on the performance data as well as the history of the car. 

---

## **4. Structure of the Website**

* Pages:
  - `index.html` -> Home page
  - `data.html` -> Performance data
  - `gallery.html` -> Image gallery
  - `history.html` -> Page about the history/ name of the car
  - (`imprint.html` -> imprint if necessary and possible within the given time)

---

## **5. Design and Layout**

- Colors: White, Orange, Black
- Tables for the `data.html` page
- Header, consistant through every page -> Orange & Black
- Footer (-> Link to imprint)

---

## **6. Milestones**

  * **Week 1:** Planning the structure, discuss division of labor
  * **Week 2:** Writing HTML pages
  * **Week 3:** Styling with CSS
  * **Week 4:** Testing and final adjustments
